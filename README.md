# Java Course Project: Community Garden Appointment Service
Java program utilized to give a user the ability to create an appointment by choosing a specific date and time, including the ability of choosing to donate.
<br>
## About
The flow of the program consist of: welcome page, contact information user input, appointment services user input, confirmation display, optional donation.
<br>
## Overview
This program contains three java files: Register.java, Date.java, Time.java. Each file contains classes that pertain to frontend and backend details that allows user to interact with the user interface and functionalities to complete an appointment confirmation.
<br>
## Tools and Components Used
JFrame: Represents a window with a title bar and borders. It is the main container used to hold GUI components.
JLabel: Used to display a single line of read-only text or an image. It's used for displaying informational messages.
JButton: Represents a clickable button. It generates an action event when clicked.
JPanel: A generic container that is invisible by default. It's used to organize components, manage layout, and provide padding and borders.
GridBagLayout: A flexible layout manager that aligns components vertically and horizontally. It allows components to be resized dynamically.
GridBagConstraints: Defines constraints for components when they are added to a GridBagLayout. Constraints include parameters like grid location, weight, insets, and fill.
JTextField: A text input component that allows the user to enter a single line of unformatted text.
ActionListener: An interface used to handle action events, such as button clicks. It contains a single method, actionPerformed(), which is invoked when an action occurs.
JCheckBox: Represents a checkbox that can be checked or unchecked. It's used for selecting multiple options from a list.
<br>
## Description
This Java application provides a step-by-step registration process for users interested in visiting a community garden. It utilizes Swing components to create an intuitive graphical interface for users to input their contact information, select an appointment date and time, specify donation preferences, and receive a confirmation of their appointment.
