# Java-Course-Project-Community-Garden-Appointment-Service
Java program utilized for user to create an appointment with a local garden.

# About
The flow of the program consist of: welcome page, contact information user input, appointment services user input, confirmation display, optional donation.

# Overview
This program contains three java files: Register.java, Date.java, Time.java. Each file contains classes that pertain to frontend and backend details that allows user to interact with the user interface and functionalities that to complete an appointment confirmation.
